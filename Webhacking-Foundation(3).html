PHP의 URL 파라미터


URL 파라미터에 값을 넣어 페이지에 반영하는 방법
GET 메소드를 이용하여 PHP 코드를 만든 다음, URL 파라미터에 형식을 맞추어서 값을 넣는다.

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    안녕하세요. <?php echo $_GET['address']; ?>에 사시는   <!-- GET 메소드 이용 -->
    <?php echo $_GET['name']; ?>님.

</html>

MySQL


MySQL 테이블의 구조

이 테이블은 2행과 4열로 이루어져 있다.

++ MySQL 테이블 생성


내가 만들 테이블 

mysql> CREATE TABLE topic(
    ->   id INT(11) NOT NULL AUTO_INCREMENT,  --  (1)
    ->   title VARCHAR(100) NOT NULL,         --  (2)
    ->   description TEXT NULL,               --  (3)
    ->   created DATETIME NOT NULL,           --  (4)
    ->   author VARCHAR(30) NULL,             --  (5)
    ->   profile VARCHAR(100) NULL,           --  (6)
    ->   PRIMARY KEY(id));                    --  (7)
  1. id 칼럼은, 식별칼럼으로서, 보통 11자리수 까지 반영하며, 빈칸이어선 안되며, 자동적으로 1씩 증가하도록 설정해야한다.
  2. title 칼럼은, 제목은 길지 않아도 되며(100), 빈칸이어선 안된다.
  3. description(본문) 칼럼은, 길어야 하므로(65,000 문자를 허용하는 TEXT), 내용이 없는 것도 허용함.
  4. create(생성시간) 칼럼은, 날짜 시간을 모두 허용하는(DATETIME)을 사용하고, 빈칸이어선 안된다.
  5. author(저자) 칼럼은, 문자인데 30글자 안으로 자르면 좋다.(VARCHAR(30))
  6. profile(프로필) 칼럼은, 길 필요가 없으니 100글자 정도로 허용하고, 빈칸 허용.

JS 변수와 대입 연산자


변수(Variable)
변수는 데이터를 저장하는 장소이며, 데이터를 읽고 수정할 수도 있다.

var name = “joho”;

대입 연산자(Operator)
대입 연산자(“=”)는 오른쪽의 값을 왼쪽의 변수에 저장한다. 대입 연산자 왼쪽에는 값을 저장할 수 있는 변수가 위치하고 대입연산자의 오른쪽에는 변수에 저장할 값이 위치한다.
